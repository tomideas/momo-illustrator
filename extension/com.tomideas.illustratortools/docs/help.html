Momo Tools v2.88

Adobe Illustrator CEP 扩展面板 · 画板 / 颜色 / 文字 / 工具 · 最后更新 2026-06-19

安装：将 com.tomideas.illustratortools 放到 ~/Library/Application Support/Adobe/CEP/extensions/，重启 Illustrator 后在 窗口 > 扩展功能 打开。

画板

复制画板 v1.1.2

复制选中的画板及其所有内容，放在现有画板正下方。

选择画板Shift 多选或输入范围（如 1-5, 8, 11-13）
名称编辑勾选「重命名复制的画板」后，查找→替换文本。例：查找 FR_ 替换为 CN_
位置与间距左右间距 / 上下间距 / 整体位移。默认自动检测

重新排列 v1.5.8

将画板按网格重新排列，可自定义每行数量和间距。

每行个数逗号分隔（3,3,2 = 第 1 行 3 个、第 2 行 3 个、第 3 行 2 个）
间距左右间距 / 上下间距（pt）
放置位置选区左上 / 某画板正下方（可偏移）

画板更名 v1.2.1

批量或单独重命名画板。4 种命名规则：

编号FR_{n} → FR_01, FR_02… 占位符：{n}=序号 {i}=画板号 {name}=原名
替换查找原名字部分 → 替换为新文本。支持正则匹配
逐行逐行输入新名称，空行保留原名
后缀在原名末尾追加固定文字。例：后缀 _v2 → FR_01_v2

颜色

颜色标签生成

选取对象后自动生成 CMYK 色块标签。可设置色块尺寸、字体、排列方向、位置等。

颜色检查

扫描文件中的填色和描边颜色，统计并标注不符合规范的颜色（如 RGB 混入 CMYK 文件）。支持文字和形状两种对象类型。

颜色库

管理多组 CMYK 颜色，支持：

提取从 Illustrator 选中对象提取颜色
增加手动输入 CMYK 值新增
点击色块应用到选中对象的填充
••• 菜单新建/重命名/删除颜色组、排序、导入/导出 JSON

文字

文字样式检查 v8.2.0

扫描文字框，找出字体、字号、行距不一致处，用橙色框标出。支持同一字号/字体/颜色分组，可选文本框内部混合检查。画板选择列表+范围输入双选模式，标注编号+需注意同图层。

文字溢出检查 v5.9.3

检测文字是否超出文本框（红色 + 标识），用红框标注。

三条规则：

M4a内容字数超出行字数（垂直溢出）
M8转点文字后内容缩短（水平溢出）
M12段落高度超出文本框（尾端空段溢出）

尾端空白 / 空段检查 v1.0.1

检查文本框末尾多余空格、Tab、空段落，用橙色框标出。

工具

网格系统 v1.0.0

在画板上绘制网格参考线。3 种类型：

瑞士网格列数/行数 + 间距。适合排版设计
等分网格N 等分 + 间距 + 可选对角线。适合海报/封面
黄金比例斐波那契数列分割 + 双轴模式。适合摄影构图

支持 px/mm/pt 单位转换、内边距 4 边同步、实时预览图层 _网格预览、一键转为 Illustrator 参考线。

批量页码

为多个画板自动添加页码文字。可设置位置、起始页数、自定义格式。

笔记 (Momo Notes)

多分页笔记面板，支持 Markdown 表格、拖曳排序、内联重命名。最多 10 个分页。

从主面板「笔记」按钮打开。需要在 Illustrator 重启后手动打开。

开发诊断工具

面板底部版本号 v2.88，连点 3 次 显示以下隐藏工具。

溢出诊断

交互式逐框诊断：自动选取每个文本框并缩放，显示脚本判断结果，询问是否有红色 +，最后输出 FP/FN 对比报告。

尾端空白诊断

交互式逐框诊断：显示尾端字符可视化（↵=Enter, ·=空格），询问是否应标记，输出对比报告。

重置所有设置

清除所有颜色库、笔记和工具偏好设置，恢复出厂状态。
